# pdfoutline

> Get outline for PDF document

## Building

Make sure you have the Xcode Command Line Tools installed.

```
$ make
```

## Usage

```bash
$ pdfoutline Northover-AArch64.pdf
AArch64 Architecture
AArch64 Backend
Testing the Backend
Interesting Curiosities
	Load-store Patterns
	Templated Operands
	Conditional Compare
Creating the Backend
Future Ideas
```
